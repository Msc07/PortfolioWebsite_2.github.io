SQL PROJECT 2

I grouped the data based on how many popular names per each year. One interesting thing I saw was the number of popular names increased throughout the years. I wonder why that is? Increase in population?

SELECT COUNT(YearofBirth) AS number, YearofBirth
FROM `portfolio-1-1.baby_names.names`
GROUP BY YearofBirth
LIMIT 10

Just to compare how many popular names were in the year 1880 vs. 2015 Data is from Kaggle.

SELECT COUNT(YearofBirth) AS number, YearofBirth
FROM `portfolio-1-1.baby_names.names`
WHERE YearOfBirth = 1880 OR YearOfBirth = 2015
GROUP BY YearOfBirth

Found the minimum number of times the most popular names were used, which was 100.

SELECT Min(Number) AS Min
FROM `portfolio-1-1.baby_names.names`

Row	Min	
1	100

Then I wondered which name had the least, but it turns out a lot of popular names had 100 uses.

SELECT Name, Number
FROM `portfolio-1-1.baby_names.names`
WHERE Number = 100
LIMIT 5

Row	 Name	 Number	
1	 Dollie   100
2	 Mathilda 100
3	 Roxie    100
4	 Lucille  100
5	 Nelle    100

In the dataset the Number 100 appears 1358 times.

SELECT COUNT(Number) AS Low_number
FROM `portfolio-1-1.baby_names.names`
WHERE Number = 100

Row	Low_number	
1	  1358

In 2015 which name had the least number of uses? A lot, so I limited it to the first 3 names.

SELECT YearOfBirth, Name, Number
FROM `portfolio-1-1.baby_names.names`
WHERE YearOfBirth = 2015 AND Number = (SELECT MIN(Number) AS Min FROM baby_names.names)
LIMIT 5

Row	YearOfBirth   Name	Number	
1	  2015       Analeah     100
2	  2015       Cristal     100
3	  2015       Fabiola     100

I think I like names that start with "O" and I was able to come up with the most popular baby girl names that start with O.

SELECT Name
FROM `portfolio-1-1.baby_names.names`
WHERE Name LIKE 'O%' AND Sex = "F"
GROUP BY Name
LIMIT 5

Row   Name	
1     Olive
2     Ollie
3     Ora
4     Ola
5     Olga

For a boy name I think I like names that start with A and end with an S.

SELECT Name
FROM `portfolio-1-1.baby_names.names`
WHERE Name LIKE 'A%s' AND Sex = "M"
GROUP BY Name
LIMIT 5

Row   Name	
1     Amos
2     Augustus
3     Aloysius
4     Alois
5     Andres
